The MIDI Event Handler is a real-time MIDI controller management interface. It listens for configured MIDI chords and maps them to structured event lifecycles (start, duration, end).

Built with FastAPI, HTMX, and Nuitka, it supports hot-pluggable devices, real-time status updates, and YAML-configurable mapping logic.

ðŸ“– User Manual

1. Upload a Mapping

Use the sidebar to drag-and-drop or upload a .yaml file that defines your event mappings.

2. Start / Stop the App

Click START to activate MIDI listening. When running, active events and MIDI status will be displayed.

3. Restart the Application

Click RESTART to trigger a full server reload (e.g. after config changes).

4. WebSocket Connection

The status LED at the bottom of the sidebar shows live connection status with the backend.

5. Event Monitoring

All triggered events will be displayed in the main status area with details about chord, type, and duration.
