## Screenshots

Screenshots are stored in the `assets/` folder (shared by both manuals since the app UI is in English).

See `assets/manifest.txt` for the full list of required screenshots.

## Adding Screenshots

1. Take screenshots at a consistent resolution (recommended: 1200px wide)
2. Save as PNG with descriptive names matching `manifest.txt`
3. Place in the `assets/` folder
4. Uncomment the corresponding `![...]()` line in the manuals
